# 📄 Product Requirement Document (PRD) — Frontend

**Project Name:** Website Ikatan Alumni SMATQ Abi-Ummi (IKRAABI)

**Tech Stack:** HTML5, Tailwind CSS (via CDN), FontAwesome / Lucide Icons, Vanilla JavaScript / Alpine.js (opsional untuk interaktivitas sederhana).

---

## 1. Overview & Objectives

* **Tujuan Website:** Menjadi wadah digital resmi untuk menghubungkan alumni SMATQ Abi-Ummi, membagikan informasi/kegiatan, database direktori alumni, serta memfasilitasi kolaborasi/kontribusi alumni.
* **Kesan Visual (Look & Feel):** Profesional, hangat, terpercaya, modern, dan bernuansa Islami-inklusif sesuai identitas sekolah.

---

## 2. Design System & Theme Configuration

### Color Palette Setup (Tailwind Config Inline)

Karena menggunakan Tailwind CSS via CDN, kita bisa menyuntikkan *color palette* kustom kamu melalui script `tailwind.config` di bagian `<head>` HTML:

```html
<script src="https://cdn.tailwindcss.com"></script>
<script>
  tailwind.config = {
    theme: {
      extend: {
        colors: {
          brand: {
            blue1: '#1D6CB4',
            blue2: '#154F92',
            blue3: '#0F356B',
            orange1: '#F07821',
            orange2: '#E25E14',
            orange3: '#C84E0C',
            gold1: '#E5BA44',
            gold2: '#C1932E',
            gold3: '#8B6214',
          }
        }
      }
    }
  }
</script>

```

### Tipografi & Peran Warna

* **Biru (`brand-blue1` s/d `blue3`):** Warna dominan untuk Header, Hero section background, Primary Button, dan Link. Memberikan kesan *trust*, ketulusan, dan profesionalisme.
* **Oranye (`brand-orange1` s/d `orange3`):** Warna aksen/highlight untuk Call to Action (CTA), badge status, efek *hover*, dan indikator penting. Menggambarkan energi, keseruan, dan kreativitas.
* **Emas (`brand-gold1` s/d `gold3`):** Digunakan secara elegan untuk aksen logo, border khusus, kartu penghargaan/prestasi alumni, atau nuansa keagamaan/kehormatan.
* **Font:** Sans-serif yang tegas & modern (misal: *Inter* atau *Plus Jakarta Sans* via Google Fonts) untuk mencerminkan identitas IKRAABI yang kuat.

  <script>
    // 1. Inisialisasi Icon Lucide
    lucide.createIcons();

    // 2. Dark Mode Toggle Script
    const themeToggleBtn = document.getElementById('themeToggle');
    const htmlTag = document.documentElement;

    // Cek preference lokal atau sistem
    if (localStorage.theme === 'dark' || (!('theme' in localStorage) && window.matchMedia('(prefers-color-scheme: dark)').matches)) {
      htmlTag.classList.add('dark');
    } else {
      htmlTag.classList.remove('dark');
    }

    themeToggleBtn.addEventListener('click', () => {
      htmlTag.classList.toggle('dark');
      if (htmlTag.classList.contains('dark')) {
        localStorage.theme = 'dark';
      } else {
        localStorage.theme = 'light';
      }
    });

    // 3. Responsive Mobile Drawer Navigation Script
    const mobileMenuBtn = document.getElementById('mobileMenuBtn');
    const closeDrawerBtn = document.getElementById('closeDrawerBtn');
    const mobileDrawer = document.getElementById('mobileDrawer');
    const drawerBackdrop = document.getElementById('drawerBackdrop');
    const mobileNavLinks = document.querySelectorAll('.mobile-nav-link');

    function openDrawer() {
      drawerBackdrop.classList.remove('hidden');
      setTimeout(() => {
        drawerBackdrop.classList.remove('opacity-0');
        mobileDrawer.classList.remove('translate-x-full');
      }, 10);
    }

    function closeDrawer() {
      drawerBackdrop.classList.add('opacity-0');
      mobileDrawer.classList.add('translate-x-full');
      setTimeout(() => {
        drawerBackdrop.classList.add('hidden');
      }, 300);
    }

    mobileMenuBtn.addEventListener('click', openDrawer);
    closeDrawerBtn.addEventListener('click', closeDrawer);
    drawerBackdrop.addEventListener('click', closeDrawer);

    // Tutup drawer jika link navigasi diklik
    mobileNavLinks.forEach(link => {
      link.addEventListener('click', closeDrawer);
    });

    // 4. Search & Dynamic Filter Alumni Script
    const searchInput = document.getElementById('searchInput');
    const angkatanFilter = document.getElementById('angkatanFilter');
    const alumniCards = document.querySelectorAll('.alumni-card');
    const noResult = document.getElementById('noResult');

    function filterAlumni() {
      const query = searchInput.value.toLowerCase().trim();
      const selectedAngkatan = angkatanFilter.value;
      let visibleCount = 0;

      alumniCards.forEach(card => {
        const nama = card.getAttribute('data-nama').toLowerCase();
        const profesi = card.getAttribute('data-profesi').toLowerCase();
        const angkatan = card.getAttribute('data-angkatan');

        const matchesQuery = nama.includes(query) || profesi.includes(query);
        const matchesAngkatan = selectedAngkatan === 'all' || angkatan === selectedAngkatan;

        if (matchesQuery && matchesAngkatan) {
          card.classList.remove('hidden');
          visibleCount++;
        } else {
          card.classList.add('hidden');
        }
      });

      // Tampilkan pesan kosong jika tidak ada hasil
      if (visibleCount === 0) {
        noResult.classList.remove('hidden');
      } else {
        noResult.classList.add('hidden');
      }
    }

    searchInput.addEventListener('input', filterAlumni);
    angkatanFilter.addEventListener('change', filterAlumni);
  </script>
</body>
